# Core Keeper サーバー設定ツール

Xserver VPS で構築された Core Keeper サーバーの設定を簡単に変更するためのコマンドラインツールです。

## 機能

- **参加人数上限の変更**: サーバーに同時に接続できるプレイヤーの最大数を設定します。
- **ワールド名の変更**: サーバー一覧に表示されるワールド名を変更します。
- **ゲームIDのリセット**: 招待用の接続 ID を新しいランダムな文字列（28文字）にリセットします。
- **ワールドモードの変更**: ノーマル、ハード、クリエイティブ、カジュアルの各モードを切り替えます。
- **ワールドスロットの変更**: 読み込むワールドのスロット番号を指定します。
- **イベントシーズンの変更**: ハロウィンやクリスマスなどの季節イベントを強制的に有効化、または自動設定にします。

## 使い方

1. Xserver VPS で Core Keeper サーバーをインストールします。
2. rootでSSHログインします。
3. 以下のコマンドを実行します。
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/shinya-blogger/xserver-vps-tools/main/corekeeper/config.sh)"
```
4. メニューに従って設定を変更してください。
5. 設定変更後、サーバーの再起動を求められます。「y」を入力すると、新しい設定でサーバーが再起動します。

## 注意事項

- このツールは Xserver VPS の Core Keeper イメージ専用に設計されています。
- **ワールドモードの変更**において、クリエイティブモードと非クリエイティブモードの相互変換はゲームの仕様上制限される場合があります。
- 設定ファイル（`ServerConfig.json`）を直接書き換えます。
